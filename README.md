# random-ethernet-address -- Make up a random MAC address

Generate a random ethernet MAC address, setting the right bits.
I couldn't find a tool like this ready & convenient, so I made one!

```
$ random-ethernet-address
66:e5:41:64:40:3d
$ random-ethernet-address
f2:d0:cc:93:1d:b0
```
